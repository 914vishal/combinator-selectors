CSS Combinators

Descendant ( )
Selects elements inside another element.
Example:

first-container p

Child (>)
Selects direct children only.
Example:

first-container > p

Adjacent Sibling (+)
Selects next element right after another.
Example:

first-container + p

General Sibling (~)
Selects all siblings after another element.
Example:

first-container ~ p
